# terraform-fabric-modules

Terraform modules for the [FABRIC](https://fabric-testbed.net) testbed.

## k8s

A single-site Kubernetes cluster (kubeadm) on a FABRIC slice, with optional
storage, load balancing, ingress, and monitoring.

- Module: [k8s/](k8s/)
- Example: [examples/k8s-cluster/](examples/k8s-cluster/)
